## Install
please rewrite your composer.json.
````
・・・
    "repositories": [
            {
                "type": "vcs",
                "url": "https://github.com/maccotsan/PHPStringTool.git"
            }
        ],
        "require": {
            "maccotsan/stringtool": "dev-master"
        }
・・・
````
